Data Analytics Project
This project is a simple Python program that performs data analysis on a CSV file using Pandas.
Features
- Load CSV files
- Clean data (remove duplicates, fill missing values)
- Filter data
- Group data
- Show summary statistics
Requirements
- Python 3.x
- pandas
How to Run
1. Open Command Prompt in the project folder
2. Run the script using:

```bash
python data_analytics.py
